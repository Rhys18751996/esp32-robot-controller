# esp32-robot-controller
A scalable ESP32 robot control framework featuring Bluetooth controller input, structured command mapping, and a non-blocking runtime designed for real-time robotics and future expansion.

## Helpful notes
install the CP210XX driver for the esp32
- https://www.silabs.com/software-and-tools/usb-to-uart-bridge-vcp-drivers?tab=downloads
- unzip the right click "silabser" then click on install. the device will show up in device manager after that.
